Sobre o Projeto

O que é o Construtor Project?

O Construtor Project é uma plataforma web desenvolvida com Python/Flask para organizar, catalogar e visualizar projetos de construção civil e seus modelos tridimensionais.

O sistema oferece um catálogo de projetos com categorias (residencial, comercial, industrial e institucional), um módulo de modelos 3D com visualização interativa via Three.js, e um banco de dados SQLite com configuração por ambiente.

Tecnologias

  • Python 3.12
  • Flask 3.1
  • Flask-SQLAlchemy
  • SQLite
  • Three.js
